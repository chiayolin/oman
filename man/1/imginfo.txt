NAME
       imginfo - Image information utility


SYNOPSIS
       imginfo [options]


DESCRIPTION
       The  imginfo  command  displays information about an image.  Please use
       the --help command line switch and the JasPer Software Reference Manual
       for more information.

       jasper(1)
