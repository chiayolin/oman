NAME
       npm-rm - Remove a package

SYNOPSIS
	 npm rm <name>
	 npm r <name>
	 npm uninstall <name>
	 npm un <name>

DESCRIPTION
       This uninstalls a package, completely removing everything npm installed
       on its behalf.

SEE ALSO
       o npm help prune

       o npm help install

       o npm help 5 folders

       o npm help config
