NAME
       jasper - File format converter specialized in JPEG-2000 encoding


SYNOPSIS
       jasper [options]


DESCRIPTION
       The jasper command converts to and from JPEG-2000 files. Please use the
       --help command line switch and the JasPer Software Reference Manual for
       more information.
