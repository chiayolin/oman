NAME
       pfbtopfa   -  Convert  Postscript  .pfb	fonts  to  .pfa  format  using
       ghostscript

SYNOPSIS
       pfbtopfa input.pfb [output.pfa]

DESCRIPTION
       This script invokes gs(1) to convert a .pfb file into a .pfa file.

SEE ALSO
       gs(1)
