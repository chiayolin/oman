NAME
       npm-prefix - Display prefix

SYNOPSIS
	 npm prefix [-g]

DESCRIPTION
       Print  the  local  prefix  to  standard out. This is the closest parent
       directory to contain a package.json file unless -g is also specified.

       If -g is specified, this will be the value of the  global  prefix.  See
       npm help 7 npm-config for more detail.

       o npm help bin

       o npm help 5 folders

       o npm help config
